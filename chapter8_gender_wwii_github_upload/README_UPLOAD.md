# Chapter 8 upload package: Gender and World War II

This ZIP contains a text-only GitHub Pages reader split into four sections:

1. Global Context
2. Japan
3. China
4. Korea

## Easy upload method

Upload the whole `chapter-08-gender-wwii` folder to the root of your existing GitHub Pages repository.

After GitHub Pages updates, open:

```text
https://YOUR-USERNAME.github.io/YOUR-REPO/chapter-08-gender-wwii/
```

This method does not overwrite your existing homepage.

## To link it from your current homepage

Add a normal link/card to:

```text
chapter-08-gender-wwii/
```

## If your current site already uses chapters.json

Use the files inside:

```text
chapter-08-gender-wwii/integration-library-format/
```

Copy this folder into your current site structure, then paste the JSON from:

```text
chapter-08-gender-wwii/integration-library-format/chapter-entry-for-chapters.json
```

into the `chapters` array of your existing root `chapters.json`.

## Audio note

No MP3 or timing data is included. This package is ready for text reading now. If you generate MP3 files later, add them to each section folder and update the `audio` fields in your existing site metadata.
